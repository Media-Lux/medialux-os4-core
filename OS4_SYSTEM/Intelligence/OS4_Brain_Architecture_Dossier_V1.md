# 2. GRUNDPRINZIPIEN DES GEHIRNS

## 2.1 Delegation statt Zentralarbeit
Das Gehirn arbeitet wie ein Team von Experten.
Supervisor + Delegation Engine entscheiden:
- Wer erledigt was?
- Welche Avatare werden eingesetzt?
- Welche Tools werden kombiniert?

## 2.2 Kostenoptimierung
@Brain entscheidet:

- Wann ChatGPT lokal genutzt wird  
- Wann OpenAI API sinnvoll ist  
- Wann Dev-KI gestartet wird  
- Wann Gemini sinnvoll wäre  
- Wann nichts ausgelagert wird  

Regel:
> „Nur API-Einsatz, wenn es objektiv schneller, besser oder günstiger ist.“

## 2.3 Vollständige Rückverfolgbarkeit
Alle Schritte werden geloggt durch:
- StepLogger  
- ZIP-Packager  
- Repo Indexer  
- Academy Extractor

## 2.4 Keine Rückentwicklung
Integrity Guard verhindert:
- redundante Module  
- doppelte Arbeit  
- Regelbrüche  
- Übergehen von alten Entscheidungen

## 2.5 Upgradefähigkeit
Jedes Modul hat eine Version:
- V1 → V2 → V3  
Das Gehirn erkennt, wann Updates nötig sind.
