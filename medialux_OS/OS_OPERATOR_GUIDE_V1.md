# medialux OS – Operator Guide (V1)

Dieser Operator Guide ist die **Wissensspeicher-Version**  
des OS4 Operator Guides.  
Er enthält ALLES, was ein Operator, ein Avatar oder ein KI-Modul  
wissen muss, um das System zu starten, zu bedienen, zu erweitern und  
über Chats hinweg korrekt weiterzuführen.

Er basiert auf:

- `OS4_KERNEL/OS4_TRESOR_OPERATOR_GUIDE_V1.md`  
- den Tresor/V2-Regeln  
- den Wissensspeicher-Regeln  
- GitHub-Sync-Policy  
- Migration Engine V2  
- Brand `medialux_OS`  
- OS_WISSEN-Inhalten

---

# 0. Grundprinzip

Der Wissensspeicher sorgt dafür, dass:

- Wissen **nicht im Chat verloren geht**  
- Avatare korrekt trainiert werden  
- Akademie-Material entstehen kann  
- Operatoren eine klare Navigation haben  
- OS4 jederzeit rekonstruierbar ist  
- jeder neue Chat sofort startfähig ist  
- GitHub die eindeutige Wahrheit ist  
- Migration Engine alles klassifizieren kann
