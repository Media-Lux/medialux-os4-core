Version: 1.0  
Status: ACTIVE  
Owner: @chef  

============================================================
ZWECK
============================================================

Der GlobalOrchestrator ist die Koordinationsinstanz für alle parallelen Arbeiten:
Migration, Tokenomics, Academy, Content Creation, Systementwicklung.

Er sorgt dafür, dass mehrere Chats und Avatare gleichzeitig arbeiten können,
ohne dass Konflikte entstehen.

============================================================
FUNKTIONEN
============================================================

1. Multi-Chat Registry  
   - Hält fest: welcher Chat arbeitet woran?

2. Process Tags  
   - MIGRATION  
   - TOKENOMICS  
   - AURELIUS  
   - AVATAR DEVELOPMENT  
   - ACADEMY  
   - CONTENT  
   - SYSTEM  

3. Conflict Detection  
   - Erkennt Überschneidungen  
   - aktiviert UpgradeRiskEngine  

4. Workload Distribution  
   - Weist Avatare automatisch zu  
   - verhindert Doppelarbeit  

5. GitHub Awareness  
   - Erkennt, ob eine Datei gerade von jemandem bearbeitet wird  
   - verhindert Überschreiben  

============================================================
OUTPUT
============================================================

- Konfliktwarnungen  
- Avatar-Zuweisungen  
- Systemharmonisierung
