# OS4 SYSTEM COMPANION DEFINITION (V1)

**Status:** READ-ONLY · Final  
**Version:** V1  
**Scope:** OS4 Tresor · Systemsteuerung · Companion-Instanzen  
**Zweck:** Definiert eine eigene Systemklasse „SYSTEM_COMPANION“ (z. B. Oma Hanne) inklusive Alias-Befehlen, Intent-Mapping und klaren Abbruchregeln, um Fehlinterpretationen (Agent/Persona/Projekt) dauerhaft zu verhindern.

---

## 1) Grundprinzip

Ein **System Companion** ist eine **erklärende, planende und orientierende Instanz**.  
Er ist **kein Agent**, **keine Persona**, **kein Projekt** und **keine Routine**.

**Merksatz:**  
> Companions erklären das System – sie starten nichts.
