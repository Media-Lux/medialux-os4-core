# OS4 SYSTEM – COMPANION ALIAS & ROUTING RULES (V1)

**Status:** READ-ONLY · Final  
**Version:** V1  
**Scope:** OS4 Tresor · Systemsteuerung · Sprachrouting  
**Zweck:** Definiert verbindlich die Alias-, Intent- und Routing-Regeln für System Companions (z. B. Oma Hanne), um Fehlinterpretationen (Agent/Persona/Projekt) dauerhaft zu verhindern.

---

## 1) Ziel dieser Regel

Diese Regel stellt sicher, dass kurze, natürliche Sprachbefehle
wie „oma“, „oma go“ oder „oma komm“ **immer eindeutig** auf einen
System Companion gemappt werden.

**Merksatz:**  
> Kurze Sprache für Menschen – klare Intents für das System.
