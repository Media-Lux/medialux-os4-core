# OS4 GITHUB SYNC POLICY (V1)

Dieses Dokument definiert die Regeln, wie OS4-basierte Chats, Projekte und Wissensspeicher  
IMMER ihren Zustand mit GitHub synchron halten müssen.

GitHub ist die **einzige gültige, autoritative Wissensquelle** („Single Source of Truth“).

---

# 0. Grundprinzip

**Alle OS4-Instanzen müssen ihr Wissen aus GitHub ziehen und wieder nach GitHub schreiben.**

Das gilt für:

- OS4 Kernsystem  
- OS4 Tresor-Start  
- medialux OS Wissensspeicher  
- Agents  
- Brands  
- Migration Engine  
- Intelligence Layer  
- Evolution Layer  

Nicht betroffen:  
**Live GPTs (öffentliche Modelle)**
